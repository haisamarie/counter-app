# カウンターアプリ

シンプルなカウンターアプリ

## 🚀 機能一覧

- カウントアップ、カウントダウン機能
- 数値リセット機能
- 上限値の設定機能

## 🏗 技術スタック

| 分類           | 技術        |
| -------------- | ----------- |
| フレームワーク | React       |
| 言語           | TypeScript  |
| スタイル       | CSS modules |
| デプロイ       | Vercel      |
